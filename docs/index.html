Welcome to Patch

Patch uses the power of iOS Automations to send text messages from your Pebble.

When you send a message, Patch sends an email to yourself with the message body and recipient details. Then, an iOS automation intercepts the email and immediately sends the text message.

The whole process takes less than ten seconds and requires no intervention after you hit send on the Pebble.

Configuration

Prerequisites

  1. A Microsoft Outlook account. Patch uses Outlook since it's free and supports push notifications to the iOS Mail app.
  2. Shortcuts installed on the iOS device you want to send messages from.

Automation Setup

  1. Add your Outlook account to the native iOS Mail app (Guide).
  2. Install the Patch iOS Shortcut. You will be asked a few questions on first time setup.
  3. Navigate to the automations tab. Create a new automation and select "Email."
  4. Configure the following:
    Automation Field Value
    Sender (Your Outlook Email)
    Subject Contains New Patch
    Account Outlook
    Recipient (Your Outlook Email)
    Run Immediately On
